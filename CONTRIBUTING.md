# Contributing Guidelines / Насоки за принос

We are excited that you want to contribute to this AI Prompt Library! Below are the guidelines for submitting new prompts.

Ние се радваме, че искате да допринесете за тази библиотека с AI промптове! По-долу са насоките за добавяне на нови промптове.

---

## How to Contribute / Как да допринесете

### 1. Structure / Структура
Every prompt should follow our structured template / Всеки промпт трябва да следва нашия структуриран шаблон:
- **Role** (e.g., Senior Developer, Copywriter) / **Роля**
- **Context & Goal** / **Контекст и цел**
- **Task Instructions** / **Инструкции за задачата**
- **Constraints & Format** / **Ограничения и формат**

### 2. Language / Език
- Please provide both English and Bulgarian versions if possible. Use the `-BG.md` suffix for Bulgarian files.
- Моля, предоставете както английска, така и българска версия, ако е възможно. Използвайте наставката `-BG.md` за българските файлове.

### 3. Submission Process / Процес на изпращане
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/new-prompt`).
3. Add your prompt to the appropriate category folder.
4. Commit your changes and push to your fork.
5. Create a Pull Request (PR) explaining the use case.
